Instalace projektu (v Linuxu):
cd pocitovamapa
python3 -m venv .venv
.venv/bin/activate
python -m pip install -r requirements.txt

Spuštění projektu:
flask --app webapp run

Jako základ pro tvorbu značek na mapě byl použit obrázek z pixabay.com
